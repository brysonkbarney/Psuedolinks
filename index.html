Link Pseudo-Classes

A pseudo-class is used to define a special state of an element. For example, it can be used to:

  • Style an element when a user mouses over it
  • Style visited and unvisited links differently
  • Style an element when it gets focus

Today we are focused on the psuedo classes for Links.

:link

A :link allows you to select the specific color for the link. Usually this class only relates to an unvisited link. The later classes void this class.

See the Pen Untitled by TheHawkk96 (@TheHawkk96) on CodePen.

:visited

A :visited class allows you to choose the color of the link after its been visited. If the link hasn't been interacted with it wont show the class.

See the Pen Untitled by TheHawkk96 (@TheHawkk96) on CodePen.

:hover

A :hover tag allows you to change the color of a link when the cursor is hovering over the link.

See the Pen :link by TheHawkk96 (@TheHawkk96) on CodePen.

:active

The :active class changes the color of the link while its active. The color changes for a brief second after you click the link. The change is small and quick.
